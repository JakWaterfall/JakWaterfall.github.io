LinkedIn Learning.

I used LinkedIn Education as further research to get myself ready for the next term. In the next term I am expected to learn the programming language C++ so to prepare myself for that I used two of LinkedIn’s most popular online courses “Learning C++” taught by Erin Colvin and “C++ Essential Training” taught by Bill Weinman.

C++ Learning covered the basics of the language and well as many ideas and concepts used in all programming languages:

  • Variables and arrays / const
  • Flow control with if statements and loops
  • Functions
  • Opening and editing files
  • Data structures

C++ Essential Training covered very briefly some of the topics taught in the former course at the start and then when in to detail of some more advanced topics within C++:

  • Basic syntax
  • Data types (including Bit fields, Enums, Unions…)
  • Operators
  • Functions (overloading, using recursion)
  • Classes and Objects
  • Templates
  • Standard Library
  • Standard Template Library
